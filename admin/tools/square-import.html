🛹 Square → 2ntr Import

Drop CSVs exported from Square Dashboard. This tool maps Square fields → our schema and upserts into Supabase. Safe to re-run — rows are matched on Square's ID and updated in place.

📦 Items / Products

Square path: Items & Orders → Items → Actions → Export library. Expected columns: Token, Item Name, Description, Price, SKU, Category, Enabled and variants.

👥 Customers

Square path: Customers → Directory → Export → All customers → CSV. Expected columns: First Name, Last Name, Email, Phone, Company, Notes, address fields, birthday.

💰 Transactions (12 months)

Square path: Reporting → Transactions → Last 12 months → Export → CSV. Historical sales only — no card data touches us.

🧑‍💼 Team members

Square path: Staff → Team → Export → CSV. This ONLY seeds the staff roster for reference — it does NOT create Supabase Auth users (that happens per-person with a real email invitation later).
